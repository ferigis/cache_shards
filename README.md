
# Cacherl

__Authors:__ Felipe Ripoll. ([`ferigis@gmail.com`](mailto:ferigis@gmail.com)).

Cacherl is a generational cache built on top of [Shards](https://github.com/cabol/shards)

## Build

    $ git clone https://github.com/ferigis/cacherl.git
    $ cd cacherl
    $ make

## Tests

    $ make tests
